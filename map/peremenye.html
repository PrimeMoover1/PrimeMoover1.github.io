Переменные

Переменной считается выделенная (зарезервированная) область памяти, предназначенная для хранения информации. Созданная переменная служит контейнером для различного типа данных. Переменная состоит из имени и выделенной области памяти.

Для объявления переменной используется ключевое слово var:

var message;

После объявления переменной (для которой мы уже придумали имя) в неё можно сохранять данные.

var message = ‘hello’;

В нашем случае в переменную messege сохранилась строка ‘hello’; Чтобы получить данные из этой переменной, нужно обратиться к ней «по имени»:

console.log(message); // выведет на экран строку «hello».

Значения в переменной не постоянны и по желанию их можно менять на протяжении всей программы:

var message = ‘hello’;
message = ‘good bye’;
console.log(message); // выведет на экран строку ‘good bye’.

Так же, значение одной переменной можно копировать в значение другой:

var message = ‘hello’;
var saySomething = message;
console.log(saySomething); // выведет на экран строку ‘hello’.

Объявление переменной через let и const.

Объявление переменной через let:

У объявления через let есть три основных отличия от var:

1. Переменная объявленная через let имеет область видимости только блок в котором она объявлена:

var x = 10;
if (x > 5)  {
var x = 15;
console.log(x); // 15
}
console.log(x) // 15;

2. Переменную let нельзя повторно объявить.

var x = 10;
var x = 15 // 15;

let x = 19;
let x = 20; // error;

3. При использовании в цикле, для каждой итерации создается своя переменная.

Объявление переменной через const:

При использовании объявления const создается константа, которую нельзя менять.

const message = ‘hello’;
message = ‘bye’; // error

Имена переменных

Имя переменной может состоять из букв, цифр, символов « $ »и « _ » . Первый символ не должен быть цифрой.

var myName; // правильное имя переменной.
var dollar$; // правильное имя переменной.
var my-name; // не правильное имя.
var 1object; // не правильное имя.

Имена переменных чувствительны к регистру. Переменная «myAlbum» и «myalbum» - две разные переменные. В качестве имени переменной нельзя использовать зарезервированные и ключевые слова JavaScript (полных список слов здесь).

Для читаемости и удобства имя переменной принято оформлять в двух стилях: camelCase (верблюжья нотация), snake_case (змеиная нотация).
